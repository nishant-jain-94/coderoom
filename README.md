# Coderoom

A Review Utility Tool for Gitlab which facilitates in code reviewing easily.

## Install

```
sudo npm install -g coderoom
```

## Command Line Usage

```
coderoom help
coderoom version
coderoom initialize
coderoom clone
coderoom members
coderoom open-issue
coderoom generate-insights
```
